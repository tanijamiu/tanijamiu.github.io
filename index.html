Balaji Fast Food Sales Report

This personal project involved building an interactive Excel dashboard to visualize and conduct an in-depth sales analysis. The goal was to examine customer behavior, sales trends, revenue generated and the impact of payment method on revenue.

You can view the dashboard for this project on GitHub

Walmart Report

This personal project involved creating an interactive Power BI report to analyze Walmart's sales. I chose the Walmart dataset because retail data offers valuable business insights. It provided a comprehensive view of Walmart’s operations, including transaction details, product performance, customer behavior, store dynamics, and external factors like weather. In this analysis, I explored product demand across different weather conditions, sales performance on weekdays versus weekends, and customer purchasing preferences.

You can view the full report on GitHub

Andy Retail Report

This project was part of my academic work, where I used MySQL to extract insights from four related tables in the Andy Retail database. The objective was to analyze sales performance across different regions, assess employee efficiency, understand customer purchasing behavior, and evaluate store profitability.

To enhance the analysis, I exported the results to Power BI and created visualizations to present my findings effectively.

You can check out my analysis and visualizations on GitHub
